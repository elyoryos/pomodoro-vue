1. Contexto del Proyecto
En la actualidad, mantener la concentración durante el estudio o el trabajo en el ordenador es un reto debido a las constantes distracciones. La técnica Pomodoro ha demostrado ser un método eficaz para mejorar la productividad mediante bloques de tiempo de trabajo concentrado alternados con breves descansos.

2. Objetivos

Crear una aplicación web de una sola pantalla (Single Page Application) que permita a los usuarios aplicar la técnica Pomodoro de forma intuitiva.

Integrar un gestor de tareas básico para que el usuario sepa en qué está trabajando durante el bloque de tiempo activo.

Ofrecer una interfaz visualmente clara que cambie de estado (color/diseño) dependiendo de si el usuario está en fase de "Trabajo" o "Descanso".

Desarrollar el proyecto utilizando Vue.js para garantizar una interfaz reactiva y un código limpio.

3. Alcance (Scope)

Incluido: * Temporizador funcional con los tiempos estándar de la técnica (25 minutos de trabajo, 5 minutos de descanso).

Controles básicos: Iniciar, Pausar y Reiniciar.

Un campo de texto para añadir la tarea actual.

Cambios visuales dinámicos según el estado del temporizador.

No incluido: * Base de datos externa o registro de usuarios (Backend).

Estadísticas históricas a largo plazo.

Notificaciones de escritorio o sonido (para mantener la simplicidad en esta primera versión).
