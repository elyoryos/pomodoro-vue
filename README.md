# 🍅 Pomodoro Vue - Spec-Driven Development

Aplicación web de temporizador Pomodoro desarrollada con Vue 3 y Vite. Este proyecto es una práctica de clase para demostrar el uso de Inteligencia Artificial aplicando la metodología Speckit (Desarrollo Orientado a Especificaciones).

## 🚀 Funcionalidades
- Temporizador Pomodoro clásico (25 min trabajo / 5 min descanso).
- Transiciones automáticas de tiempo y color de fondo según el estado.
- Gestor de la tarea actual integrado.
- Interfaz reactiva (Single Page Application).

## 📂 Documentación de Especificación (Speckit)
En la raíz de este repositorio puedes encontrar los documentos de la fase de especificación:
1. `foundations.md`: Contexto y objetivos.
2. `specify.md`: Especificación funcional y de interfaz.
3. `planning.md`: Tareas y decisiones técnicas.

## 🛠️ Instalación y ejecución local
Si deseas ejecutar este proyecto en tu ordenador:

1. Clona el repositorio.
2. Instala las dependencias: `npm install`
3. Arranca el servidor local: `npm run dev`
